AI agent safety

Polyshield

Approval, policy, and audit infrastructure for AI coding agents using local MCP tools.

Polyshield started from a practical problem: coding agents are no longer just writing suggestions, they can touch files, git, browsers, databases, and local services. I am building the control layer between the agent and those tools so risky actions can be inspected, approved, logged, and constrained before they execute.

  • Built a local runner that pairs with the Polyshield gateway and launches stdio MCP servers.
  • Designed outbound polling so the user's machine does not need inbound ports or tunnels.
  • Added sandbox-oriented controls: scoped working directories, stripped environments, timeouts, snapshots, and dry-run diffs for filesystem-confined calls.
  • Modeled the product around policies, human approvals, audit trails, credential isolation, and prompt-injection defenses.

UI systems

JSwingX

A custom Java Swing framework inspired by Roblox-style UI composition.

JSwingX treats Swing less like a set of native widgets and more like a drawable scene graph. I built primitives for frames, images, text, text boxes, component containers, layout helpers, z-ordering, anchor positioning, rich text, image caching, selection systems, and tween animations.

  • Implemented Dim and Dim2-style scale/offset sizing for responsive Java desktop interfaces.
  • Built a tree-based UI model with named instances, parent-child composition, and reusable primitives.
  • Documented the framework so someone familiar with Roblox UI can understand and use it quickly.

Browser extension

BetterLyrics

Synced lyrics and cleaner music UI for Spotify Web.

BetterLyrics is a Chrome extension that brings a Beautiful/Spicy Lyrics-style experience to Spotify Web. It works in a Chrome-only mode and can optionally connect to a Spicetify companion bridge for stronger token access.

  • Integrated multiple lyrics providers with fallback order for synced, syllable, line, static, and plain lyrics.
  • Built a companion bridge mode without making it required for the main extension.
  • Added debug helpers and logging that avoid leaking full access tokens or bridge keys.

Engineering competition

Pumps & Pipes

Grand Winner for an RC ChemE car combining chemistry, electronics, and robotics.

My team built a ChemE car with an iodine clock reaction tied into a light-detection circuit and motor control. I worked on electronics, programming, motors, sensors, and debugging the integration between chemistry, hardware, and code.

  • Used a visual chemical reaction as part of the stopping mechanism.
  • Connected photoresistor/light detection to microcontroller-driven motor behavior.
  • Iterated under competition pressure across hardware and software failure modes.
